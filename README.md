# Nohooks
Nohooks is a project to add webhooks to popular web services using [Convoy](https://github.com/frain-dev/convoy). The service is also hosted at [app.nohooks.io](https://app.nohooks.io)

## Requirements
- Postgres
- Redis
- Convoy
